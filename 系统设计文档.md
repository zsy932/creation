# 餐厅菜品推荐系统设计文档

## 一、技术架构

### 1.1 整体架构

本系统采用**纯前端静态架构**，基于现代Web技术栈构建，具有轻量级、易部署、高性能的特点。

```
┌─────────────────────────────────────────┐
│              用户界面层                  │
│        (HTML + CSS + JavaScript)        │
└─────────────────────────────────────────┘
                    │
                    ▼
┌─────────────────────────────────────────┐
│              业务逻辑层                  │
│         (App.js - 事件处理)             │
└─────────────────────────────────────────┘
                    │
                    ▼
┌─────────────────────────────────────────┐
│              数据层                      │
│    (HTML模板 + 内置数据 + LocalStorage) │
└─────────────────────────────────────────┘
```

### 1.2 技术栈

#### 核心技术
- **HTML5**: 语义化页面结构
- **CSS3**: 现代化样式设计（Flexbox、Grid、动画）
- **JavaScript (ES6+)**: 原生JavaScript，无框架依赖
- **Tailwind CSS**: 实用优先的CSS框架

#### 辅助技术
- **Remix Icon**: 图标库
- **CDN资源**: Tailwind CSS在线加载
- **图片服务**: Picsum Photos占位图
- **静态图片**: 外部图片CDN

#### 构建工具
- **Vite**: 现代化构建工具（用于开发服务器和打包）
- **Node.js**: JavaScript运行环境

### 1.4 数据流设计

```
用户操作 → 事件监听 → DOM操作 → 数据更新 → UI刷新
```

**数据存储策略：**
- **静态数据**: 硬编码在HTML/JavaScript中
- **用户偏好**: 使用LocalStorage本地存储
- **临时状态**: 使用JavaScript变量维护
