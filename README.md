# AI智能点餐助手

一个基于AI的智能菜品推荐系统，能够根据用户的个人历史点餐数据、口味偏好及营养标签，快速从上千种组合里精准推荐菜品。

## 🚀 功能特色

### 🧠 AI智能推荐
- **个性化推荐**: 基于用户历史订单和口味偏好进行智能分析
- **营养均衡**: 结合健康目标推荐合适的菜品
- **时间感知**: 根据当前时间段推荐适合的餐品
- **热门推荐**: 基于销量和评分推荐热门菜品

### 🎯 精准推荐场景
- "今天你最想吃的菜" - 基于个人偏好的精准推荐
- "上周吃太油这周推荐吃虾仁饺子" - 营养均衡建议
- "减脂餐推荐超级碗" - 健康目标导向
- 快速午餐、健康轻食等场景化推荐

### 📊 智能分析
- 口味偏好分析（辣度、甜度、咸度）
- 营养摄入分析（卡路里、蛋白质、脂肪等）
- 饮食习惯分析（菜系偏好、食材偏好）
- 健康目标匹配（减脂、增肌、低糖等）

## 🔧 推荐算法

### 多维度分析
1. **历史数据分析**: 分析用户过去的点餐记录
2. **口味偏好识别**: 识别用户对辣度、甜度等的偏好
3. **营养需求匹配**: 根据健康目标推荐合适菜品
4. **时间场景感知**: 根据时间段推荐适合的餐品

### 推荐类型
- **个性化推荐**: 基于个人偏好的专属推荐
- **健康推荐**: 符合健康目标的营养搭配
- **热门推荐**: 大家都喜欢的经典选择
- **场景推荐**: 特定场景下的智能推荐
